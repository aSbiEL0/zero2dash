Place the startup GIF in this directory as `startup.gif`, or override the path with `BOOT_SELECTOR_GIF_PATH`.
Assets in this directory should be named `mainmenu.png` for the 4-quadrant menu, `day-night.png` for the day/night chooser, `yes-no.png` for shutdown confirmation, and `credits.gif` for the info animation. You can still override them with `BOOT_SELECTOR_MAIN_MENU_IMAGE`, `BOOT_SELECTOR_DAY_NIGHT_IMAGE` (or legacy `BOOT_SELECTOR_IMAGE_PATH`), `BOOT_SELECTOR_SHUTDOWN_IMAGE`, and `BOOT_SELECTOR_INFO_GIF`.
