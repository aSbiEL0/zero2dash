Place the startup GIF in this directory as `startup.gif`, or override the path with `BOOT_SELECTOR_GIF_PATH`.
`credits.gif` is still kept here as a shared boot asset, but it is no longer used by the normal Credits button flow.
Assets in this directory should be named `mainmenu.png` for the 4-quadrant menu, `day-night.png` for the day/night chooser, `yes-no.png` for shutdown confirmation, and `keypad.png` for the PIN keypad. You can still override them with `BOOT_SELECTOR_MAIN_MENU_IMAGE`, `BOOT_SELECTOR_DAY_NIGHT_IMAGE` (or legacy `BOOT_SELECTOR_IMAGE_PATH`), `BOOT_SELECTOR_SHUTDOWN_IMAGE`, `BOOT_SELECTOR_KEYPAD_IMAGE`, and `BOOT_SELECTOR_INFO_GIF`.
