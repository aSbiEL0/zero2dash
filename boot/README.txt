Place the startup GIF in this directory as `startup.gif`, or override the path with `BOOT_SELECTOR_GIF_PATH`.
Assets in this directory should be named `mainmenu.png` for menu page 1, `mainmenu-page2.png` for menu page 2, `day-night.png` for the day/night chooser, `yes-no.png` for shutdown confirmation, `keypad.png` for the PIN keypad, and `credits.gif` for the info animation. You can still override them with `BOOT_SELECTOR_MAIN_MENU_IMAGE`, `BOOT_SELECTOR_MAIN_MENU_PAGE2_IMAGE`, `BOOT_SELECTOR_DAY_NIGHT_IMAGE` (or legacy `BOOT_SELECTOR_IMAGE_PATH`), `BOOT_SELECTOR_SHUTDOWN_IMAGE`, `BOOT_SELECTOR_KEYPAD_IMAGE`, and `BOOT_SELECTOR_INFO_GIF`.
The selector uses a paged layout: page 1 bottom-left launches the standalone NASA/ISS app, page 2 bottom-left opens Photos, and the left strip flips between menu pages.
