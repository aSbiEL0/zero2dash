# Historical Plan

`PLAN.md` is retained as a historical record of the shell-first migration plan that introduced:

- the shell-first runtime
- Dashboards as a child app
- Photos as a dedicated slideshow app
- timer-driven shell mode switching

The active rebuild plan is now [`rebuild-plan.md`](./rebuild-plan.md).

Use `PLAN.md` only for historical context when reviewing how the shell-first baseline was delivered.
